## Setup

1. Install [node](http://nodejs.org/).

2. Run ```npm update```.

3. Install [grunt-cli](http://gruntjs.com/getting-started#installing-the-cli): ```npm install -g grunt-cli```

4. Install [bower](http://bower.io/): ```npm install -g bower```

5. Run ```bower update```.

## Developing

1. Run ```grunt server```.
